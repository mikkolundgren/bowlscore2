Saved Scores

How to Use:

  • Enter pins knocked down: Type numbers 0-10 for each throw
  • Strike: Enter "X" or "10" in the first throw of a frame
  • Spare: Enter "/" in the second throw when total pins = 10
  • Gutter ball: Enter "0" or "-"
  • Save Score: Click "Save Score" to store each series in the database separately
  • Load Scores: Click "Load Saved Scores" to view all stored games
  • Delete Score: Delete scores directly from the list
